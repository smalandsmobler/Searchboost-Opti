| Problem | Antal |

| Sidor som returnerar 4xx-fel (= borta för Google) | **34** |
| Sidor i sitemap som ger 4xx | **22** |
| Sidor i sitemap som time-outar | **8** |
| Sidor utan meta description | **14** |
| Sidor utan H1-rubrik | **20** |
| Bilder som saknar alt-text | **77** |
| Bilder som ger 4xx-fel (trasiga) | **77** |
| Duplicerade sidtitlar | **9** |
| Mobil PageSpeed-poäng | **60/100** |

**34 produktsidor som Google inte kan indexera.** Det innebär att potentiella kunder som söker på de produkterna hittar noll — inte ens ett felmeddelande. De hittar er konkurrent.

**77 trasiga produktbilder** på en sajt som säljer båtar för 5 miljoner kronor. Det är inte ett intryck som säljer.

**Mobilpoäng 60** — under gränsen för vad Google belönar i sökresultaten.

Ni har ett starkt produktutbud. Men sajten presterar långt under sin potential.
