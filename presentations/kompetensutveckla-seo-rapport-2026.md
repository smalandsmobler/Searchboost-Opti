## 1. Executive Summary

Kompetensutveckla.se är en **stark sajt** med DIR 50, 2 199 referring domains och 7 907 backlinks. Sajten rankar redan i top 10 för 36 av 50 övervakade sökord i GSC — men genererar bara **37 klick/månad** på dessa sökord trots enorma impressions (9 000+).

**Huvudproblem:** Extremt låg CTR (0.4%) trots bra positioner. Orsaker:
1. Dåliga meta-titlar som inte lockar klick
2. 142 trasiga URL:er (971 förekomster av brutna länkar)
3. Kaotisk kategoristruktur med kannibalisering
4. 200+ gratisresurser begravda och osynliga för Google

**Potentialen är enorm** — med optimerade titlar, fixade redirects och synligt kunskapsinnehåll kan sajten gå från 37 klick/mån till 500+ klick/mån inom 3 månader.

## 3. Tekniska problem

### 3.1 Trasiga URL:er — KRITISKT

**142 unika URL:er** med totalt **971 förekomster** av brutna länkar.

**Orsak:** URL-strukturen ändrades från:
```
/arbetsmiljoarbete/kunskapsbanken/X
→ /vara-tjanster/arbetsmiljoarbete/kunskapsbanken/X
```
Utan 301-redirects. 121 av 142 = Kunskapsbanken-artiklar.

**Konsekvens:**
- Google crawlar 404-sidor → tappar trust
- Besökare landar på felmeddelanden → bounce rate ökar
- Intern länkjuice (link equity) läcker ut
- Kunskapsbanken rankar sämre än den borde

### 3.2 Kategoristruktur — Kannibalisering

**Samma utbildning finns på 3+ ställen:**
- BAM: /webbutbildningar/, /fysiska-utbildningar/, /bam-utbildning/
- Ledarskap: 4 olika platser
- SAM: /kurser-2/ och /sam-utbildning/

**Google vet inte vilken sida som är viktigast** → kannibalisering → alla sidor rankar sämre.

### 3.3 Kunskapsbanken & Gratismaterial — Osynligt

200+ resurser (checklistor, mallar, rutiner, AFS-dokument) är begravda under `/vara-tjanster/arbetsmiljoarbete/...` — 3-4 nivåer djupt. Ingen navigering från huvudmenyn. PDF:er oindexerade av Google.

### 3.4 Meta-data

- **Meta-titlar:** Existerar men ej optimerade för CTR. Saknar CTA och power words.
- **Meta-descriptions:** Många auto-genererade eller saknas helt.
- **Schema markup:** Minimal. Saknar Course, FAQPage, Organization schema.

## 5. Redirect-lista — 142 trasiga URL:er

### Mönster och fix

Huvudmönstret: alla kunskapsbanks-URL:er behöver redirectas:

```
GAMMAL: /arbetsmiljoarbete/kunskapsbanken/{artikel}
NY:    /vara-tjanster/arbetsmiljoarbete/kunskapsbanken/{artikel}
```

### Kategorier av trasiga URL:er

| Typ | Antal | Mönster |
|-----|-------|---------|
| Kunskapsbanksartiklar | 121 | /arbetsmiljoarbete/kunskapsbanken/X → /vara-tjanster/... |
| AFS-regler | 12 | /arbetsmiljoarbete/afs/X → /vara-tjanster/... |
| Kurs-sidor | 5 | /kurser-2/X (ändrad URL) |
| Rutiner/mallar | 4 | /arbetsmiljoarbete/rutiner/X → /vara-tjanster/... |

### Implementation

Två alternativ:

**Alt A: WordPress Rank Math / Redirection plugin (rekommenderat)**
- Lägg in varje redirect i Rank Math → Redirections
- 301 Permanent redirect
- Kan göras i bulk via CSV-import

**Alt B: .htaccess-regel (snabbt)**
```apache
# Redirect alla /arbetsmiljoarbete/ till /vara-tjanster/arbetsmiljoarbete/
RedirectMatch 301 ^/arbetsmiljoarbete/(.*)$ /vara-tjanster/arbetsmiljoarbete/$1
```

**Rekommendation:** Alt B fixar 90% av redirectsen med EN rad. Alt A för de enstaka URL:er som inte matchar mönstret.

## 7. CTR-optimering — Omedelbar effekt

Den största "quick win" är att förbättra meta-titlar och descriptions. Nuvarande CTR på 0.4% är absurt lågt för positioner i top 10.

### Exempel — Före/Efter

**"bam utbildning" (pos 5.3, 231 imp, 1 klick = 0.4% CTR)**

Före:
```
BAM — Bättre arbetsmiljö 2 dagar | Kompetensutveckla
```

Efter:
```
BAM Utbildning — Bättre Arbetsmiljö 2 dagar ✓ Online & klassrum | Boka nu
```

**"arbete på väg" (pos 7.4, 495 imp, 3 klick = 0.6% CTR)**

Före:
```
Arbete på väg | Kompetensutveckla
```

Efter:
```
Utbildning Arbete på Väg — Giltig i 5 år ✓ Online & lärarlett | Boka
```

**"sam utbildning" (pos 4.6, 53 imp, 2 klick = 3.8% CTR)**

Före:
```
SAM — Systematiskt arbetsmiljöarbete | Kompetensutveckla
```

Efter:
```
SAM Utbildning — Systematiskt Arbetsmiljöarbete ✓ 1 dag online | Boka idag
```

### Regler för CTR-optimerade titlar

1. Huvudsökord först i titeln
2. Unik USP (giltighetstid, format, dagar)
3. Checkmark (✓) som visuell signal
4. CTA i slutet (Boka, Läs mer, Ladda ner)
5. Max 60 tecken (visas inte om längre)

## 8. Schema Markup — Rekommendationer

### Course Schema (alla kurssidor)
```json
{
  "@type": "Course",
  "name": "BAM — Bättre Arbetsmiljö",
  "description": "...",
  "provider": {
    "@type": "Organization",
    "name": "Kompetensutveckla"
  },
  "courseMode": ["online", "onsite"],
  "duration": "P2D",
  "offers": {
    "@type": "Offer",
    "price": "3995",
    "priceCurrency": "SEK"
  }
}
```

### FAQPage Schema (kurssidor + kunskapsbank)
Genererar FAQ-snippet i Google → mer yta i sökresultaten → högre CTR.

### Organization Schema (startsidan)
```json
{
  "@type": "Organization",
  "name": "Kompetensutveckla",
  "url": "https://kompetensutveckla.se",
  "contactPoint": {...}
}
```

### BreadcrumbList Schema (alla sidor)
Visar tydlig navigering i Google → ökad trust och CTR.
