## Identifierade problem — sammanfattning

### Kritiska (Hög prioritet)

| Problem | Antal | Påverkan |
|---------|-------|----------|
| **Description missing** | 50 sidor | Google visar slumpmässigt utdrag istället för kontrollerat budskap |
| **H1-tagg saknas** | 39 sidor | Google förstår inte sidans ämne |
| **JavaScript ej minifierad** | 69 resurser | Långsam laddning |
| **JavaScript ej cachad** | 69 resurser | Onödig bandbredd varje besök |
| **CSS ej komprimerad** | 69 resurser | Långsam rendering |
| **CSS ej minifierad** | 69 resurser | Uppblåst sidstorlek |
| **CSS ej cachad** | 69 resurser | Upprepade nedladdningar |
| **Alt-text saknas på bilder** | 24 sidor | Missar bildsök-trafik + tillgänglighet |
| **Bilder för stora** | 12 sidor | Långsam mobil + sämre Core Web Vitals |

### Strukturella

| Problem | Antal | Påverkan |
|---------|-------|----------|
| **4XX HTTP-statuskoder** | 2 sidor | Besökare hamnar på felsidor |
| **Redirect-kedja** | 1 URL | Långsammare crawlning |
| **Duplicerade sidtitlar** | 5 sidor | Google vet inte vilken sida som ska rankas |
| **Title för kort** | 4 sidor | Missar sökord i title-taggen |
| **Title för lång** | 2 sidor | Klipps i sökresultaten |
| **Duplicerad H1** | 5 sidor | Förvirrar Google om sidans ämne |
| **Multipla H1-taggar** | 4 sidor | Otydlig sidhierarki |
| **Sitemap saknas i robots.txt** | 1 | Google hittar inte alla sidor effektivt |
| **Sidor utan inlänkar** | 6 sidor | "Föräldralösa" sidor som Google knappt hittar |
| **Interna länkar till redirects** | 4 st | Slösar "link juice" |
| **Extern länk till 3XX** | 3 st | Dålig UX + signal |

### Innehåll & URL-struktur

| Problem | Detalj |
|---------|--------|
| **Blogginlägg under /uncategorized/** | 15+ inlägg har URL:en `/uncategorized/namn/` — ser oprofessionellt ut och ger noll SEO-värde |
| **Gamla sidor med siffror som URL** | /uncategorized/5592/, /uncategorized/4462/, /uncategorized/6300/ — ingen människa eller sökmotor förstår |
| **Testsidor indexerade** | /project/testsida/, /arbetssida/ — borde ej synas |
| **Author- och kategorisidor indexerade** | /author/admin/, /category/uncategorized/ — tunn/duplicerat innehåll |
| **Inga blogginlägg sedan april 2025** | Sajten uppdaterad feb 2026 men bloggen är tyst |

## Vad vi gör — 3-månaders SEO-plan

### Månad 1: Teknisk SEO & grunden

| Åtgärd | Detalj |
|--------|--------|
| **Installera Rank Math Pro** | SEO-plugin som ersätter Divi:s begränsade SEO-funktioner |
| **Meta descriptions på alla 50 sidor** | Unika, keyword-optimerade beskrivningar |
| **H1-taggar på alla 39 sidor** | Tydlig hierarki med rätt sökord |
| **Title-optimering** | Fixa duplicerade, för korta och för långa titlar |
| **Sitemap → robots.txt** | Så Google hittar alla sidor direkt |
| **Fixa 4XX-sidor** | Redirect eller ta bort de 2 trasiga URL:erna |
| **Rensa redirect-kedjan** | Direkt-redirect istället för kedja |
| **Rensa testsidor** | noindex på /arbetssida/, /project/testsida/ |
| **Fixa /uncategorized/ URL:er** | Flytta blogginlägg till vettiga kategorier |
| **noindex på tunn innehåll** | /author/admin/, /category/uncategorized/ etc. |

### Månad 2: Innehåll & on-page

| Åtgärd | Detalj |
|--------|--------|

| **Tjänstesidorna optimeras** | Varje tjänstesida får unikt fokus-keyword, optimerad rubrikstruktur, intern länkning |
| **Intern länkstrategi** | Koppla ihop tjänstesidor → projekt → blogg |
