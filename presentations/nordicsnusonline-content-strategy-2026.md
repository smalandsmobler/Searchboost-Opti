Nuvarande struktur har parallella kategorier som kannibaliserar varandra:

```
NUVARANDE (kaotiskt):
/sv/snus/                          ← 266 produkter (ALLA)
/sv/portionssnus/                  ← Subset
/sv/kop-snus-online/vitt-snus/     ← Vitt snus
/sv/kop-snus-online/kop-vit-portion-snus-online.../  ← Vit portion
/sv/snussorter/                    ← Annu en oversikt
/sv/mini-portion/                  ← Mini-format
/sv/snus-med-smak/                 ← Smak-baserad

FORESLAGEN (rensat):
/sv/snus/                          ← Huvudkategori
  /sv/snus/portionssnus/           ← Underkategori
  /sv/snus/vitt-snus/              ← Underkategori (tobaksfritt)
  /sv/snus/lossnus/                ← Underkategori
  /sv/snus/mini-portion/           ← Underkategori
```

## DEL 4: FAQ- och Guidesidor att skapa

### Nya sidor som behover byggas

Utover bloggartiklarna behover sajten dedicerade guide-/FAQ-sidor som permanenta kunskapshubbar.

#### 1. Guider-hub: /sv/guider/

En oversiktssida som samlar alla guider. Fungerar som topical authority-sida.

**Innehall:**
- Kort intro (100 ord)
- Lankgrid till alla guider (12+ artiklar)
- Kategoriserat: Nyborjarguider, Varumarkesguider, Jamforelser, Kopguider

#### 2. Utokad FAQ: /sv/vanliga-fragor/

Nuvarande FAQ ar grund. Behover utvidgas med minst 25 fragor grupperade i kategorier.

**Fragor att lagga till:**

*Bestallning & Leverans:*
- Hur langer tar leveransen?
- Vilka betalningsalternativ finns?
- Kan jag bestalla snus till Norge/Danmark/Finland?
- Hur fungerar aldersverifieringen?
- Vad kostar frakten?

*Produkter & Val:*
- Vad ar skillnaden mellan vitt snus och portionssnus?
- Vilken nikotinstyrka ska jag valja?
- Vad ar nikotinpasar?
- Vilken snus ar bast for nyborjare?
- Hur lange haller en pase snus?

*Forvaring & Anvandning:*
- Hur forvarar jag snus?
- Hur anvander man snus?
- Kan man svalja snus?
- Hur lange ska man ha snuset under lappen?

*Halsa & Regler:*
- Ar snus farligt?
- Ar nikotinpasar sakerare an snus?
- Aldersgrns for snus i Sverige?
- Vad sager forskningen om snus?

**Schema:** Varje fraga ska ha FAQ-schema (structured data) for Google rich snippets.

#### 3. Om oss-sida: /sv/om-oss/ (E-E-A-T-kritisk)

Nuvarande "Om oss"-information ar minimal. For en YMYL-sajt (tobak + e-handel) ar detta affärskritiskt.

**Innehall:**
- Foretagshistoria (VATT AB, Lidkoping)
- Team/grundare (namn, bakgrund)
- Kvalitetslofte (GothiaTek-standard, kylkedja)
- Kontaktuppgifter (adress, telefon, e-post, organisationsnummer)
- Returpolicy och kundnojdhetsgaranti

## DEL 5: Internlankningsstrategi

### Principer

1. **Silo-struktur**: Varje innehallstyp lankar primärt inom sin silo
2. **Pyramidmodell**: Startsida → Kategorier → Varumarken → Produkter → Guider
3. **Kontextuella lankar**: Varje bloggartikel lankar till 3-5 relevanta produkter
4. **Breadcrumbs**: Implementera BreadcrumbList-schema pa alla sidor

### Lankflode

```
STARTSIDA (kopa snus online)
    |
    ├── KATEGORISIDOR (portionssnus, vitt snus, lossnus...)
    |       |
    |       ├── VARUMARKESSIDOR (ZYN, VELO, General, Knox...)
    |       |       |
    |       |       └── PRODUKTSIDOR (ZYN Spearmint Mini, VELO Tropical Mango...)
    |       |
    |       └── [Korslankar till relaterade kategorier]
    |
    ├── GUIDER/BLOGG
    |       |
    |       ├── Nyborjarguide → lankar till: kategorier + specifika produkter
    |       ├── Varumarkesguider → lankar till: varumarkessidor + produkter
    |       ├── Jamforelser → lankar till: bada varumarken + produkter
    |       └── Kopguider → lankar till: hela sortimentet + fraktinfo
    |
    └── FAQ → lankar till: guider + kategorier + kundservice
```
