## Identifierade problem — sammanfattning

### Kritiska (Hög prioritet)

| Problem | Antal | Påverkan |
|---------|-------|----------|
| **Inget SEO-plugin installerat** | Hela sajten | Ingen kontroll över metadata, sitemap eller schema |
| **Meta descriptions saknas** | Alla sidor + produkter | Google visar slumpmässigt utdrag istället för kontrollerat budskap |
| **H1-tagg saknas** | Många sidor | Google förstår inte sidans ämne |
| **Title-taggar för korta/generiska** | Många sidor | Missar sökord och klickfrekvens |
| **Schema markup saknas** | Alla sidor | Inga rikare sökresultat (pris, betyg, lagerstatus) |
| **Alt-text saknas på bilder** | Alla produktbilder | Missar bildsök-trafik + tillgänglighet |
| **Inga interna länkar** | Mellan kategorier | Google kan inte följa sidstrukturen effektivt |

### Strukturella

| Problem | Antal | Påverkan |
|---------|-------|----------|
| **Bara 2 blogginlägg** | 2 st | Nästan inget innehåll för Google att indexera utöver produkter |
| **"Under bearbetning"-sida indexerad** | 1 sida | Oprofessionellt + slösar crawl-budget |
| **Produktsidor ej SEO-optimerade** | 159 produkter | WooCommerce-standardtitlar utan sökord eller beskrivningar |
| **Kategorisidor ej optimerade** | 24 kategorier | Inga unika beskrivningar eller metadata |
| **Ingen sitemap i robots.txt** | 1 | Google hittar inte alla sidor effektivt |

### Innehåll & Sökbarhet

| Problem | Detalj |
|---------|--------|
| **159 produkter utan SEO-metadata** | Produkterna saknar optimerade titlar, beskrivningar och schema — osynliga för Google |
| **24 kategorier utan beskrivningar** | Kategorierna "Soffor", "Sängar", "Matbord" etc. har inga unika texter — tunt innehåll |
| **Inga lokala SEO-signaler** | "Möbelaffär Mora" rankar #1 men ingen strukturerad lokal info (schema, Google Business) |
| **0 klick trots impressions** | Sajten syns men ingen klickar — meta descriptions och titlar måste förbättras |

## Vad vi gör — 3-månaders SEO-plan

### Månad 1: Teknisk SEO & grunden

| Åtgärd | Detalj |
|--------|--------|
| **Installera Rank Math Pro** | SEO-plugin med full WooCommerce-integration (produkt-schema, metadata, sitemap) |
| **Meta descriptions på alla 15 sidor** | Unika, keyword-optimerade beskrivningar för varje sida |
| **Meta descriptions på nyckelkategorier** | Soffor, Sängar, Matbord, Utemöbler, Soffbord — de viktigaste kategorierna |
| **H1-taggar på alla sidor** | Tydlig hierarki med rätt sökord |
| **Title-optimering** | Sökordsfokuserade, rätt längd, unika per sida |
| **Sitemap → robots.txt** | Så Google hittar alla 159 produkter + kategorier direkt |
| **Ta bort "Under bearbetning"-sida** | noindex eller redirect — ska ej synas i Google |
| **Grundläggande intern länkning** | Koppla ihop startsida → kategorier → produkter |

### Månad 2: Produkter & kategorier

| Åtgärd | Detalj |
|--------|--------|
| **Kategori-SEO (alla 24 kategorier)** | Unika beskrivningar och metadata för Soffor, Sängar, Matbord, Utemöbler etc. |
| **WooCommerce produkt-schema** | Product-schema med pris, lagerstatus, betyg på alla produkter |
| **Alt-text på produktbilder** | Relevanta sökord på alla produktbilder — bildsök-trafik |
| **Intern länkstrategi** | Koppla ihop kategorier sinsemellan: Vardagsrum → Sovrum → Matrum |
| **Produkttitlar optimeras** | Nyckelprodukter får sökordsfokuserade titlar |
| **Breadcrumbs-schema** | Tydlig navigationsstruktur för Google |

### Månad 3: Innehåll, lokal SEO & tillväxt

| Åtgärd | Detalj |
|--------|--------|
| **Nytt blogginnehåll** | 2–4 artiklar: "Välj rätt soffa", "Inred vardagsrummet", "Utemöbler-guide" etc. |
| **Lokal SEO för Mora** | LocalBusiness-schema, optimera för "möbelaffär Mora/Dalarna" |
| **Geografisk SEO** | Rikta mot "möbler Dalarna", "möbelaffär Falun", närliggande regioner |
| **Prestandaoptimering** | Bildkomprimering, caching, laddningstid — WooCommerce-specifikt |
| **Uppföljning & rapport** | Positioner, trafik, konverteringar — fullständig genomgång |
