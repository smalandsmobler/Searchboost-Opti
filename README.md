# Babylovesgrowth

Minimal scaffold added: TypeScript + Express example with tests, linting, Dockerfile and GitHub Actions CI.

Quick commands:

- Install deps: `npm ci`
- Run in dev: `npm run dev`
- Build: `npm run build`
- Run built: `npm start`
- Test: `npm test`
- Lint: `npm run lint`
- Build Docker image: `docker build -t babylovesgrowth .`

This scaffold was added on branch `feat/scaffold-node-ts` for review.
