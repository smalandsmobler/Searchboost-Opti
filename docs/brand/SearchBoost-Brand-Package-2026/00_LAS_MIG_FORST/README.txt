SEARCHBOOST — GRAFISK PROFIL 2026
=====================================

Detta paket innehåller allt du behöver för att bygga material
som ser ut som Searchboost — utan att det blir slop.

VAD ÄR HÄR
----------
01_logotyp/       Logotyp-varianter (ljus, mörk, ikon)
02_farger/        brand.css — färgvariabler, typografi, spacing, komponenter
03_mallar/        E-postmallar, Looker Studio-spec, deck-template
04_guider/        Sociala medier, presentation, dokument
05_dokument/      Komplett grafisk profil (huvuddokument)
06_visuell_referens/   Visuell referens på live-användning

NÄR DU BÖRJAR
-------------
1. Läs 05_dokument/grafisk-profil-searchboost.md — det är källa-till-sanning
2. För webb/email: importera 02_farger/brand.css
3. För kund-mail: kopiera 03_mallar/email-veckorapport.html som start
4. För deck: följ 04_guider/presentation-guide.md

ANTI-SLOP-REGLER
----------------
- Aldrig generisk template-känsla — varje element ska vara medvetet
- Mörk navy (#05050F) är default-bakgrund — inte vit
- Pink-magenta-gradient som accent, ej överanvänd
- Inter Tight för rubriker, Inter för brödtext
- 8px grid — allt spacing är multipler av 8
- Skuggor är subtila — aldrig hårda glas-effekter på text
- Emoji bara om kunden själv använder dem först
