## 5. DNS-INSTRUKTION

**Problem:** `feroxkonsult.se` (utan www) visar FS Datas parkeringssida.

**Lösning (FS Data / Loopia-kontrollpanel):**
1. Ta bort befintlig A-record för `@` (apex)
2. Lägg till Shopify-redirect: `ALIAS` eller `A-record` för `@` → Shopifys IP (151.101.1.57 + 151.101.65.57 + 151.101.129.57 + 151.101.193.57)
3. Eller: Lägg till `301 redirect` i FS Data-panelen: `feroxkonsult.se` → `https://www.feroxkonsult.se`

**Shopify DNS-krav (i Shopify Admin → Domains):**
- `www` CNAME → `shops.myshopify.com`
- `@` (apex) A-records → Shopifys 4 IP-adresser ovan
