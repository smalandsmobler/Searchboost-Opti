**Instruktion till Mikael:**
1. Skapa kundkonton via API innan du skickar mailen:
   ```
   POST https://opti.searchboost.se/api/portal/users
   Headers: X-Api-Key: <din-api-nyckel>
   Body: { "email": "<kundmail>", "customerId": "<id>", "password": "Searchboost2026!" }
   ```
2. Fyll i **[LÖSENORD]** med `Searchboost2026!` (eller det du väljer).
3. Skicka från mikael@searchboost.se — en mail per kund, en i taget.

## API-kommandon — Skapa kundkonton

Kör dessa innan du skickar mailen. Ersätt `<API-NYCKEL>` med din nyckel från SSM.

```bash
BASE="https://opti.searchboost.se"
KEY="<API-NYCKEL>"
PW="Searchboost2026!"

for CUST in \
  "mattias@mobelrondellen.se|mobelrondellen" \
  "arnman@phvast.se|phvast" \
  "peter.vikstrom@ilmonte.se|ilmonte" \
  "mikael.n@kompetensutveckla.se|kompetensutveckla" \
  "jakob@tobler.se|tobler" \
  "patrik.carlsson@traficator.se|traficator" \
  "mikael@smalandskontorsmobler.se|smalandskontorsmobler" \
  "mail@wedosigns.se|wedosigns" \
  "Andreas.Sternberg@feroxkonsult.se|ferox"
do
  EMAIL=$(echo $CUST | cut -d'|' -f1)
  ID=$(echo $CUST | cut -d'|' -f2)
  curl -s -X POST "$BASE/api/portal/users" \
    -H "X-Api-Key: $KEY" \
    -H "Content-Type: application/json" \
    -d "{\"email\":\"$EMAIL\",\"customerId\":\"$ID\",\"password\":\"$PW\"}" | python3 -m json.tool
  echo "--- $ID klar ---"
done
```
