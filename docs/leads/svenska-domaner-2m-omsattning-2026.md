## Källor och metodik

### Datakällor använda
1. **qualify-checkpoint.json** — 859 domäner crawlade 2026-02-22, sitemap-storlek som proxy för bolagsstorlek
2. **allabolag.se** — Verifierade omsättningssiffror för 12 bolag (Tier 1)
3. **vaning18.se** — Kundlogotyper identifierade (Notar, Ekstrands, Praktikertjänst, Södra, Monark, Kakeldax)
4. **intendit.se/referenser/** — Referenslista (27 kunder, mestadels småföretag)
5. **lankanalys-2026.md** — Backlink-data (Våning 18: 387 RD, 725K BL)
6. **smk-marknadsforing-analys-2026.md** — Kompetitordata med stora aktörer (Kinnarps, AJ Produkter)

### Begränsningar
- SE Ranking API returnerar HTTP 400 för alla domäner — organisk trafik = 0 för alla
- Omsättningsdata hämtad från allabolag.se (senaste bokslut, 2023-2024)
- Sitemapstorlek korrelerar ej alltid med omsättning (inflexus.se: 13,881 sidor men 346 KSEK omsättning)
- Kontaktuppgifter från crawl kan vara inaktuella

### Nästa steg
1. Kontakta Tier 1 med personaliserat analysmail (använd befintliga 28 PDF-analyser om tillgängligt, annars generera ny)
2. För Våning 18-kunder (ekstrands.com prioritet): Kör konkurrentanalys av deras nuvarande SEO-tillstånd
3. Komplettera Tier 3-listan med allabolag-verifiering via bulk-script
