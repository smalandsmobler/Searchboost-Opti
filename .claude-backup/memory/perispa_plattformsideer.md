# Perispa Plattformsidéer (prioriterad)

Beslut 2026-04-17: Systemet säljs INTE (struken white-label). Allt fokus = recurring kunder.

## NEED-TO-HAVE (löser problem vi har idag)

### 1. Opti Content Engine
Claude skriver artikel → Perispa publicerar direkt i WP med layout, bilder, schema, Rank Math.
**Värde:** 2-3 artiklar/kund/vecka utan manuellt arbete.

### 2. Opti Onboarding Wizard
Kund fyller i formulär → WP app-password genereras → GSC-access begärs → ABC-keywords föreslås → allt i SSM.
**Värde:** Onboarding 5 min istället för 2h.

### 3. Opti Site Doctor
Daglig health-check: broken links, 404, plugin-uppdateringar, säkerhetshål. Auto-fix + snapshots.
**Värde:** Proaktiv drift istället för att kunder ringer.

### 4. Opti Meta Writer
Bulk-metadata via Perispa: hitta sidor utan meta → Claude skriver → Perispa publicerar.
**Värde:** 10x snabbare metadata-optimering.

## NICE-TO-HAVE (säljer mer / differentierar)

### 5. Opti Page Builder
"Bygg landningssida för kontorsstolar" → Claude designar → Perispa bygger i WP.
**Värde:** 5000-15000 kr/sida utan manuellt bygge.

### 6. Opti Competitor Mirror
Crawla konkurrenters struktur → generera motsvarande sidor.
**Värde:** Programmatisk SEO driven av konkurrentinsikt.

### 7. ~~STRUKEN — White-Label~~
Systemet säljs inte. Det är vår moat.

### 8. Opti A/B Test Engine
Duplicera sida → ändra rubrik/CTA via Perispa → mät via GA4.
**Värde:** 3000 kr/test.

### 9. Opti Translation Hub
Artikel .se → .no/.dk/.fi-version med lokalisering + hreflang.
**Värde:** Nordisk expansion för kunder.

### 10. Opti Review Responder
Google-recensioner → Claude svarar i kundens tonalitet → publicera.
**Värde:** 1000 kr/mån tillägg.

## Byggordning (rekommendation)
Vecka 1: Content Engine (#1) — störst impact
Vecka 2: Meta Writer (#4) — snabbast ROI
Vecka 3: Onboarding Wizard (#2) — skalbarhet
Vecka 4: Site Doctor (#3) — kundnöjdhet
